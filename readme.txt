SAMURAI'S QUEST
---------------

Instructions:

To compile and run:
	1) Open a terminal window
	2) Set the current directory to the game's root directory. (the directory this readme is in)
	3) Run the following two commands:
		javac -sourcepath "./src" -d "./out" src/Controller.java
		java -classpath "./out" Controller
		NOTE: If these commands do not work and you are using Windows, try using '\' instead of '/'!

Gameplay instructions will be provided in the game.
